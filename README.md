# Tita — App de Recordatorios

Tu asistente de recordatorios con alma de abuela. Carinosa, atenta y siempre pendiente de vos.

## Funciones

- Recordatorios por voz y texto en espanol natural
- Medicaciones con seguimiento (frecuencia y fecha de fin)
- Cumpleanos y fechas especiales con recurrencia anual
- Alarmas confiables que suenan con la app cerrada o el telefono reiniciado
- Vista calendario mensual con indicadores por categoria
- Categorias inteligentes: compras, salud, trabajo, personal, hogar, finanzas
- Home organizada con secciones inteligentes
- Personalidad unica y carinosa
